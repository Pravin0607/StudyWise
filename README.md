EduGen is an AI-powered student-teacher portal designed to revolutionize the learning and assessment experience. The platform empowers teachers to create dynamic exams effortlessly by leveraging advanced LLMs (Large Language Models) to process and understand study materials. Students can access these exams, receive instant feedback, and track their learning progress, all within a seamless, user-friendly environment.

With features like AI-generated exams, adaptive learning suggestions, detailed performance analytics, and collaborative tools, EduGen simplifies education for teachers and enhances learning outcomes for students. By integrating cutting-edge technologies like LangChain and Ollama, EduGen delivers a personalized and efficient educational experience for all.
